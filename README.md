# Jogo do Número Secreto

Projeto criado no início do meu aprendizado de lógica de 
programação na Alura.

## Como funciona
O jogo solicita ao usuário que tente adivinhar um número 
aleatório gerado entre 1 e 10, dando dicas se o número 
informado é maior, menor ou igual ao gerado.

## Tecnologias utilizadas
- JavaScript — lógica e geração dos números aleatórios
- HTML — estrutura dos campos exibidos na tela
- CSS — estilização visual da interface
